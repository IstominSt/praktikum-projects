# Исследование рынка видеоигр

Анализируем продажи интернет-магазина компьютерных игр и игр для консолей за прошедшие года. Выявляем определяющие успешность игры закономерности. В отчете указал перспектиные для закупа жанры и платформы компьютерных игр для разных регионов в предстоящем сезоне.

Используемые инструменты: python, pandas, seaborn, визуализация данных.
