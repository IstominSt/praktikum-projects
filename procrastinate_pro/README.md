# Исследование причин неудачной рекламной компании

Расчитал бизнес метрики и динамику их изменения во времени. Исследовал причину неудачных рекламных компаний.

Выяснили: Найдены три неокупаемые рекламы, которые снижают общий показатель ROI на 120 пунктов. Даны общие рекомендации по дальнейшим действиям с этими каналами. 

Используемые инструменты: python, pandas, matplotlib, визуализация данных.
