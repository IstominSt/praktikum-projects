# Исследование рынка общественного питания Москвы

На основании анализа рынка заведений общественного питания в Москве. Был найден потенциально прибыльный район для открытия собственной кофейни. По результатам анализа подготовлена презентация для инвесторов.

Используемые инструменты: python, pandas, matplotlib, seaborn, plotly, folium, визуализация данных.
